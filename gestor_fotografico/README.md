# Gestor de Coberturas Fotográficas
Aplicación de consola desarrollada en Python para gestionar coberturas fotográficas.

## Descripción
Este proyecto permite registrar, consultar y analizar coberturas fotográficas realizadas para distintos clientes y tipos de eventos.
Fue desarrollado como práctica de programación utilizando listas, funciones, estructuras de control y búsqueda de información.

## Funcionalidades
- Agregar nuevas coberturas.
- Ver todas las coberturas registradas.
- Buscar coberturas por palabra clave.
- Ver estadísticas generales:
    - Cantidad total de coberturas.
    - Cantidad total de fotografías.
    - Coberturas agrupadas por tipo de evento.
    - Promedio de fotografías por cobertura.

## Tecnologías utilizadas
- Python 3
- Funciones
- Listas
- Bucles for y while
- Condicionales
- Busqueda y procesamiento de datos
